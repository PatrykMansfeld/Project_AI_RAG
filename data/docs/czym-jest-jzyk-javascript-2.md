# Język Javascript
Język Javascript to skryptowy język programowania, który jest używany do tworzenia interaktywnych stron internetowych oraz aplikacji mobilnych. Został stworzony przez Brendana Eich w 1995 roku i jest własnością firmy Mozilla.

## Historia powstania
Język Javascript został stworzony jako dodatek do przeglądarek internetowych, aby umożliwić tworzenie interaktywnych stron internetowych. Początkowo nazywał się "Mocha", ale później zmieniono nazwę na Javascript. W 1997 roku Netscape Communications wydała specyfikację języka Javascript, która stała się standardem dla przeglądarek internetowych.

## Podstawowe cechy
Język Javascript jest skryptowym językiem programowania, co oznacza, że nie wymaga kompilacji przed wykonaniem. Jest to dynamiczny język, który może być uruchamiany w czasie wykonywania programu. Jego podstawowe cechy to:

* Niezależność od platformy: Javascript jest językiem wieloplatformowym, co oznacza, że może być używany na różnych systemach operacyjnych.
* Interakcja z użytkownikiem: Javascript umożliwia tworzenie interaktywnych stron internetowych, które mogą reagować na działania użytkownika.
* Obsługa obiektów: Javascript obsługuje obiekty jako podstawową jednostkę danych.

## Zastosowania
Język Javascript jest używany w wielu dziedzinach, takich jak:

* Tworzenie stron internetowych: Javascript jest używany do tworzenia interaktywnych stron internetowych, które mogą reagować na działania użytkownika.
* Aplikacje mobilne: Javascript jest używany do tworzenia aplikacji mobilnych, takich jak gry i aplikacje biznesowe.
* Programowanie serwerowe: Javascript jest używany do tworzenia aplikacji serwerowych, takich jak Node.js.

## Warianty języka
Język Javascript ma kilka wariantów, takich jak:

* ECMAScript: to jest standardowy wariant języka Javascript, który jest używany przez większość przeglądarek internetowych.
* ActionScript: to jest wariant języka Javascript, który jest używany do tworzenia aplikacji Flash.
* JScript: to jest wariant języka Javascript, który jest używany przez Internet Explorer.

## Podsumowanie
Język Javascript jest skryptowym językiem programowania, który jest używany do tworzenia interaktywnych stron internetowych oraz aplikacji mobilnych. Jego podstawowe cechy to niezależność od platformy, interakcja z użytkownikiem i obsługa obiektów. Jest używany w wielu dziedzinach, takich jak tworzenie stron internetowych, aplikacje mobilne i programowanie serwerowe.
