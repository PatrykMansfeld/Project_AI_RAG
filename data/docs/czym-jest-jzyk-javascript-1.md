# Język Javascript
Język Javascript to skryptowy język programowania, który jest używany do tworzenia interaktywnych stron internetowych oraz aplikacji mobilnych. Został stworzony przez Brendana Eich w 1995 roku dla firmy Netscape Communications.

## Historia powstania
Język Javascript został pierwotnie nazwany Mocha, ale później zmieniono mu nazwę na LiveScript. W 1997 roku firma Netscape zdecydowała się zmienić nazwę języka na Javascript, aby uniknąć konfliktu z firmą Sun Microsystems, która posiadała prawa do nazwy Java.

## Podstawowe cechy
Język Javascript jest skryptowym językiem programowania, co oznacza, że nie wymaga kompilacji przed uruchomieniem. Jest to dynamiczny język, który umożliwia tworzenie interaktywnych stron internetowych oraz aplikacji mobilnych.

### Podstawowe elementy
Język Javascript składa się z następujących podstawowych elementów:

* **Zmienne**: są to wartości przechowywane w pamięci komputera.
* **Funkcje**: są to bloki kodu, które wykonują określone zadania.
* **Obiekty**: są to zbiory danych i metod, które umożliwiają przechowywanie i manipulowanie danymi.

### Właściwości
Język Javascript posiada następujące właściwości:

* **Dynamiczność**: język jest dynamiczny, co oznacza, że nie wymaga kompilacji przed uruchomieniem.
* **Interaktywność**: język umożliwia tworzenie interaktywnych stron internetowych oraz aplikacji mobilnych.
* **Wieloplatformowość**: język może być używany na różnych platformach, w tym na komputerach stacjonarnych i mobilnych.

## Zastosowania
Język Javascript jest szeroko stosowany w wielu dziedzinach:

### Strony internetowe
Język Javascript jest używany do tworzenia interaktywnych stron internetowych, takich jak strony z animacjami i efektami specjalnymi.

### Aplikacje mobilne
Język Javascript jest używany do tworzenia aplikacji mobilnych, takich jak gry i aplikacje biznesowe.

### Serwery
Język Javascript jest używany do tworzenia serwerów, takich jak serwery web i serwery aplikacyjne.

## Podsumowanie
Język Javascript to skryptowy język programowania, który jest szeroko stosowany w wielu dziedzinach. Posiada dynamiczność, interaktywność oraz wieloplatformowość, co czyni go idealnym wyborem dla tworzenia interaktywnych stron internetowych i aplikacji mobilnych.
